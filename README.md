# OzzyRenos

A modern, SEO-optimized website built for a home renovation company in the Niagara Region.

## Tech Stack

- Next.js (App Router)
- React, TypeScript, Tailwind CSS
- SEO setup with dynamic sitemap, metadata, and Open Graph
- Deployed via Vercel

## Features

- Responsive, accessible design
- Page for each service, as well as Gallery and Contact
- Contact form
- Elfsight Widget for Google reviews feature
- Dynamic sitemap and robots.txt for SEO
